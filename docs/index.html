Power

Probability of rejecting \(H_0\) when it is true is type I error (falsely reject), aka significance of the test aka \(\alpha\). Probability of not reject H_0 when it is false is a type II error, \(\beta\). Power of test is \(1-\beta\).

Confidence Intervals

Confidence set with a confidence coefficient of \(1-\alpha\) is the set.

One sided H_0

\[\begin{equation} H_0 : \mu = \mu_0 vs. H_1: \mu > Mu_0\\ T = \frac{\sqrt(x-\mu_0)}{\sigma} \sim t distribution with n-1 degrees of freedom \\ Reject H_0 if t > t_{n-1,1-\alpha} \\ Type I error: \alpha = P_{\mu_o}(T>t_{n-1,1-\alpha}) \\ Acceptance region: T \le t_{n-1,1-\alpha}\\ 1- \alpha = P_{\mu_0}(\frac{\sqrt{n}(\bar{X}-\mu_0)}{\hat\sigma}) \le t_{n-1,1-\alpha}\\ = P_{\mu_0}(\bar{X} - t_{n-1,1-\alpha}\frac{\hat\sigma}{\sqrt{n}} \le \mu_0) \end{equation}\]
